## 작성 코드
*첫 제출*
```js
function solution(s) {
    var answer = '';
    let splited = s.split(" ");    
    let converted = splited.map(e => Number(e));
    let min = Math.min(...converted);
    let max = Math.max(...converted);
    
    answer = `${min} ${max}`;
    return answer;
}
```

*메소드 체이닝으로 엮은 방식. 다소 비효율적*
```js
function solution(s) {
    var answer = s
        .split(" ")
        .map(e => Number(e))
        .reduce((acc, e) => {
	        return  [[...acc[0], e]]
		}, [[]])
        .map(e => {
            return { 
	            min: Math.min(...e), 
	            max: Math.max(...e) 
			};
        })
        .map(e => `${e.min} ${e.max}`)
        [0];
    return answer;
}
```
