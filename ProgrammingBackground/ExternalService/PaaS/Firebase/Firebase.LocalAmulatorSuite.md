# 개요
Firebase를 이용한 서비스 개발 과정에서 불필요한 비용이나 잘못된 영향 전파를 막고 안전하게 개발할 수 있도록 로컬 환경에서 Firebase 서비스를 모방해 대행해주는 개발 보조 도구이다.
