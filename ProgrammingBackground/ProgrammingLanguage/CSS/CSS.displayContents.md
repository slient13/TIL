# 정렬 관련
```css
{
	display: flex; // flex 형태로 표현
	flex-direction: column; // flex 방향
	justify-content: center; // 각 요소 정렬 (가로 방향)
	align-items: center; // 각 요소에 배정된 칸에 대해 위치 배정.	
}
```
