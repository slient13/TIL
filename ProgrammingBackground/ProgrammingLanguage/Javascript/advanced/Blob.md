# 개요
*Binary Large Object*의 약자로 **Javascript**에서 이미지, 사운드, 비디오와 같은 멀티미디어 데이터를 다룰 때 사용할 수 있는 특수 객체이다.

비슷한 용도로 쓰이는 **File**이라는 객체도 **Blob**을 바탕으로 구축된 것이다.
